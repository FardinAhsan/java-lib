Changes

Version 9.0
2013-05-30

 

Changes

OS X

  • Adds support for OS X 10.8 Mountain Lion.

JVM

  • Adds support for J2SE7 from Oracle. The native Quaqua library for this JVM has the name libquaqua64.dylib.
  • Removes support for J2SE5.
  • Removes support for org.jdesktop.layout API.
  • Removes support for PPC.

Known Issues with J2SE7 from Oracle:

  • Popup menus are not translucent.
  • JSheet is shown and hidden without a transition animation.
  • JProgressBar is not animated.

Fixes

  • Issue 133: Adds additional text translations in French, German, Italian and Swedish.

JTabbedPane

  • JTabbedPane.indexAtLocation(x,y) now properly reports the tab index when all tabs are collapsed into a single JComboBox.
